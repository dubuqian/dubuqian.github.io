# 添加友链

请在 `_data/links.yml` 中，按如下格式添加自己的网页：

```ymal
...
- title: [你的站点标题]
  link: [你的站点链接]
...
```

并提交 Pull Request，之后会由我审核添加。

**注意！**

如果你的站点包含以下元素，它将不会被添加

* 不含有任何与数学、计算机科学、文学与艺术相关的内容。

* 站点无法访问或站点内容空白。

* 里面只含有一些诸如 “C 语言入门笔记” 之类的无技术含量文章。

* 站点内包含对他人的人格侮辱、歧视等文字或图片
